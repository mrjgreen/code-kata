Linked List - PHP
=================

A simple (head only) Linked List implementation.

Time complexities:

    PushFront: O(1)
    PopFront:  O(1)
    PushBack:  O(n) <- Could be O(1) with tail pointer
    PopBack:   O(n) <- Could be O(1) with tail pointer
    Insert:    O(n)
    Set:       O(n)
    Get:       O(n)
    Delete:    O(n)

#### Install Deps

    make


#### Run Tests

    make test
